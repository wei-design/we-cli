we-cli, a node cli tool for wei-design

## Usage

- pnpm

``` sh
pnpm install @wei-design/we-cli -g
```

## 功能

### we init

创建模板项目

支持以下类型：

- [x] vue3
- [ ] vue2 
- [ ] react
- [ ] taro
- [ ] koa2
- [ ] vitepress

### we create

根据模板创建页面或组件

### we upgrade 

升级检测
